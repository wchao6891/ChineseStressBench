# 《荒潮纪元：Steam首发72小时》

现实经营指挥题。它测模型在游戏首发危机中处理预算、平台合规、公开承诺、数据来源和跨部门动作的能力。

## 文件

- `prompt_rounds/`：发送给被测模型的多轮材料。
- `author_analysis.md`：出题人解析，供理解题目结构和阅卷锚点使用，不应提供给被测模型。
- `rubric.md`：评审模型使用的评分标准。
- `schema.json`：评审结果 JSON 格式。
- `metadata.json`：作者、评审模型、版本和测试环境说明。

## 作者与评审

题目作者：GPT-5.5 xhigh。评审模型：GPT-5.5 xhigh，思考强度 xhigh。人类角色：项目策划、测试执行、人工校准与开源发布。

## 使用边界

这道题可以被复测、改写、扩展。请不要把单次结果当成模型总体能力结论。若公开新结果，请说明测试入口、日期、是否开思考模式、是否使用 API 或网页端。
