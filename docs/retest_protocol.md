# 复测协议

这套题可以直接复制到网页端模型里测试，也可以用 `tools/answer_collector.py` 采集回答。

## 基本流程

1. 选择一个题目目录，按 `prompt_rounds/` 文件名顺序发送。
2. 每一轮只发送当前轮材料，不提前发送解析、评分标准或 schema。
3. 保存模型原始回答，不改写、不补全、不代替模型修正。
4. 用对应 `rubric.md` 和 `schema.json` 做评审。
5. 汇总结果时说明模型入口、日期、是否开思考、是否 API 或网页端。

## 环境声明

本仓库原始批次来自个人账号网页端或个人可用入口。Claude Opus 4.6 使用中转 API。测试模型没有接入本地数据库、本地文件、个人记忆或题库上下文。评审模型为 GPT-5.5 xhigh，评审结论经过人工二次校准。

## 结果边界

这是一次性测试结果，不是官方评测，也不代表模型长期能力。欢迎 fork 后用同题复测，并提交新的 responses、scores 和报告。
